# Impeccable Design Context — Shopsugiツ

## Design Context

### Users
**Presenteadores premium** — pessoas buscando presentes unicos e especiais para quem amam. Chegam ao site com intencao de encontrar algo que surpreenda, que tenha historia e que transmita cuidado. A interface deve evocar **confianca e encantamento**: a sensacao de estar numa galeria curada onde cada peca foi escolhida a dedo.

### Brand Personality
**Elegante, Artesanal, Calorosa** — Sofisticacao com toque humano. Nao e uma boutique fria e distante, nem uma feira popular desorganizada. E o meio-termo perfeito: como entrar numa galeria acolhedora onde o artesao te recebe com um cafe. A marca celebra a filosofia **Kintsugi** — beleza nas imperfeicoes, ouro nas rachaduras, valor no que e feito a mao.

### Aesthetic Direction
- **Visual tone:** Warm minimalism com toques organicos e detalhes em ouro
- **References:** Etsy (curadoria artesanal, comunidade) + Aesop (tipografia elegante, espacamento generoso, premium sem ostentacao)
- **Anti-references:** Marketplaces genericos (Shopee/AliExpress), templates de e-commerce padrao, excesso de cores saturadas, layouts densos com banners piscando
- **Theme:** Light mode only — base quente (#FDFAF5 warm-50), sem dark mode
- **Language:** Portugues brasileiro, tom acolhedor e poetico

### Typography
- **Body:** DM Sans (sans-serif) — legivel, moderno, amigavel
- **Display:** Playfair Display (serif) — elegancia, personalidade, headings e precos
- **Scale pattern:** Headings usam `font-extrabold`, body usa `font-medium`, labels usam `text-xs uppercase tracking-widest`
- **Line height:** `leading-relaxed` para corpo, `leading-tight` para headings grandes

### Design Principles

1. **Warmth over coldness** — Cada pixel deve transmitir calor humano. Preferir tons quentes, bordas suaves (rounded-2xl), sombras sutis. Nunca usar cinza puro (#gray) — sempre warm neutrals.

2. **Generous breathing room** — Espacamento generoso como a Aesop: secoes com `py-24 sm:py-32`, padding interno amplo em cards (`p-8 sm:p-10`). O espaco vazio e intencional e comunica premium.

3. **Gold as punctuation, not noise** — O ouro kintsugi e usado como acento, nao como cor dominante. Aparece em CTAs, dividers, hovers, badges — nunca como background de secoes inteiras. Menos e mais.

4. **Motion with purpose** — Animacoes existem para guiar o olhar e criar encantamento (scroll-reveal, shimmer, badge-pop), nao para distrair. Easing consistente `cubic-bezier(0.22, 1, 0.36, 1)`. Sempre respeitar `prefers-reduced-motion`.

5. **Craft in the details** — Assim como o artesao cuida de cada detalhe da peca, a interface cuida de cada micro-interacao: o som sutil ao adicionar ao carrinho, o SVG de crack dourado entre secoes, a animacao do badge. Cada detalhe conta uma historia.

### Component Patterns
- **Buttons:** `rounded-full` para CTAs primarios e secundarios, `rounded-2xl` para acoes contextuais (add to cart inline)
- **Cards:** `rounded-2xl bg-white border border-warm-100 shadow-sm` com hover `-translate-y-1` e shadow upgrade
- **Dividers:** KintsugiDivider SVG component (crack dourado animado) entre secoes — identidade visual unica
- **Icons:** Inline SVG, `strokeWidth={1.5}` para consistencia, cor `text-kintsugi-500`
- **Feedback:** react-hot-toast (top-right), sons sutis no carrinho, loading spinners em botoes
- **Accessibility:** Skip nav, aria-labels, semantic HTML (sections + aria-labelledby), focus rings em `ring-kintsugi-500`

### Spacing Scale (Tailwind tokens in use)
- Section vertical: `py-20 sm:py-28` a `py-24 sm:py-32`
- Card padding: `p-6` (compact) a `p-8 sm:p-10` (feature cards)
- Grid gaps: `gap-5` a `gap-8`
- Max width: `max-w-7xl` (content), `max-w-4xl` (hero text), `max-w-3xl` (CTA sections)

### File Structure
- Layouts: `resources/js/Layouts/` — PublicLayout.tsx, AdminLayout.tsx
- Public components: `resources/js/Components/Public/` — ProductCard, HeroBanner, Footer, etc.
- Shared components: `resources/js/Components/Shared/` — Button, KintsugiDivider, Logo, Spinner, SkeletonLoader
- Admin components: `resources/js/Components/Admin/`
- Hooks: `resources/js/hooks/` — useCartSound, useScrollReveal
- Types: `resources/js/types/` — shared.ts, public.ts, admin.ts
- Styles: `resources/css/app.css` — @theme tokens, keyframes, kintsugi classes
